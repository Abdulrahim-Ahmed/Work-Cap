1. Introduction

Welcome to the Multi Currency Partner Ledger user guide. This document will help you understand how to use the module effectively to generate partner ledger reports in multiple currencies.

Note: The Multi Currency Partner Ledger module is designed for Odoo 18 and requires the Accounting module to be installed.

2. Accessing the Module

To access the Multi Currency Partner Ledger module:

Step 1: Log in to your Odoo instance.

Step 2: Navigate to the Accounting module by clicking on "Accounting" in the main menu.

Step 3: In the Accounting menu, go to "Reporting" and then click on "Multi Currency Partner Ledger".

3. Generating Reports

Follow these steps to generate a Multi Currency Partner Ledger report:

Step 1: Set the date range for your report using the "Start Date" and "End Date" fields.

Step 2: Select the target moves:

  • All Posted Entries: Only includes journal entries that have been posted.
  • All Entries: Includes all journal entries, including draft entries.

Step 3: Select partners for the report:

  • You can select specific partners or leave this field empty to include all partners.
  • Use the search function to quickly find partners by name or code.

Step 4: Select accounts:

  • You can select specific accounts or leave this field empty to include all accounts.
  • Typically, you'll want to include receivable and payable accounts.

Step 5: Select currencies:

  • Choose the currencies you want to include in the report.
  • You must select at least one currency.

Step 6: Configure additional options:

  • Show Initial Balance: Toggle this option to show or hide initial balances.
  • Reconciled: Toggle this option to show only reconciled entries.

Step 7: Click one of the following buttons:

  • View: Display the report in the browser.
  • Print: Generate a PDF report.
  • Export Excel: Download an Excel (XLSX) version of the report.

4. Understanding the Report

The Multi Currency Partner Ledger report is organized as follows:

Report Header

The header section includes:

  • Company name and information
  • Report title
  • Date range
  • Filter criteria applied

Partner Information

For each partner, the report displays:

  • Partner name
  • Partner code
  • Contact information (if available)

Currency Sections

The report is organized by currency, with each currency having its own section containing:

  • Currency name and symbol
  • Initial balance (if enabled)
  • Transaction details
  • Subtotals for the currency

Transaction Details

For each transaction, the report shows:

  • Date
  • Journal
  • Reference
  • Label/Description
  • Debit amount
  • Credit amount
  • Running balance

Totals and Subtotals

The report includes:

  • Subtotals per currency
  • Subtotals per partner
  • Grand total across all partners and currencies

5. Exporting Reports

The Multi Currency Partner Ledger module allows you to export reports in two formats:

PDF Export

To export the report as a PDF:

Step 1: Configure your report parameters as described in the "Generating Reports" section.

Step 2: Click the "Print" button.

Step 3: A PDF file will be generated and downloaded to your computer.

Excel Export

To export the report as an Excel file:

Step 1: Configure your report parameters as described in the "Generating Reports" section.

Step 2: Click the "Export Excel" button.

Step 3: An XLSX file will be generated and downloaded to your computer.

Note: The Excel export provides additional flexibility for data analysis and manipulation.

6. Tips and Best Practices

Optimizing Report Performance

  • Narrow Date Ranges: Use specific date ranges to improve report generation speed.
  • Select Specific Partners: When possible, select only the partners you need rather than generating reports for all partners.
  • Limit Currencies: Include only the currencies you need to analyze.

Effective Filtering

  • Use Partner Tags: Organize partners with tags to make selection easier.
  • Save Favorite Filters: Save commonly used filter combinations as favorites for quick access.
  • Combine with Other Reports: Use the Multi Currency Partner Ledger in conjunction with other accounting reports for comprehensive analysis.

Warning: Generating reports with many partners, accounts, and currencies over a wide date range may take longer to process.

7. Frequently Asked Questions

Why are some transactions missing from my report?

Transactions may be missing for several reasons:

  • They fall outside the selected date range
  • They involve partners or accounts not included in your filter
  • They are in currencies not selected for the report
  • If "All Posted Entries" is selected, draft entries will not appear

Why do the amounts in different currencies not match when converted?

Currency conversion depends on the exchange rates defined in Odoo. Discrepancies may occur due to:

  • Exchange rate fluctuations over time
  • Missing exchange rates for specific dates
  • Rounding differences

Ensure your exchange rates are up-to-date for accurate reporting.

Can I customize the report layout?

The standard report layout is fixed, but you can:

  • Export to Excel for custom formatting
  • Contact your Odoo developer to customize the QWeb templates if needed

How can I analyze data from multiple periods?

To compare data across multiple periods:

  • Generate separate reports for each period
  • Export to Excel and use pivot tables for analysis
  • Consider using Odoo's dashboard features for visual comparisons
