Multi Currency Partner Ledger

Generate partner ledger reports with multiple currencies

Key Features

  • Multiple Currency Support - Generate partner ledger in any currency
  • Flexible Filtering - Filter by partner, date range, and account
  • PDF & Excel Export - Export reports in different formats
  • Detailed Transaction View - See all transaction details with running balance
  • Initial Balance Option - Include or exclude initial balances

How It Works

  1. Access the Report - Go to Accounting → Reporting → Multi Currency Partner Ledger
  2. Set Parameters - Select partners, currencies, date range, and other filters
  3. Generate Report - Click "View" to see the report in the browser
  4. Export Options - Use "Print" for PDF or "Export Excel" for XLSX format

Benefits for Your Business

  • Improved Multi-Currency Management - Easily track receivables and payables in any currency
  • Better Decision Making - Understand your financial position across different currencies
  • Time Saving - Avoid manual currency conversion calculations
  • Enhanced Reporting - Get comprehensive financial reports for partners in multiple currencies
  • Excel Export - Analyze data further in Excel with the XLSX export feature

Technical Features

  • Odoo 18 Compatible - Fully compatible with Odoo 18
  • XLSX Report Support - Generate Excel reports using the report_xlsx module
  • PDF Report Support - Generate PDF reports with QWeb templates
  • Modular Design - Easy to extend and customize
  • Security - Respects Odoo security model and access rights
